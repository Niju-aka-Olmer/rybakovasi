# План деплоя — rybakovasi.ru (Astro + TinaCMS)

## Цель
Задеплоить сайт на бесплатный хостинг, настроить TinaCMS через Tina Cloud, чтобы можно было редактировать контент онлайн.

---

## Шаг 1 — GitHub
- Создать репозиторий на GitHub
- Залить текущий код
- (нужен логин/пароль от GitHub)

## Шаг 2 — Tina Cloud
- Зарегистрироваться на https://tina.io/cloud/ (free tier)
- Привязать GitHub-репозиторий
- Получить токены: `TINA_CLIENT_ID` и `TINA_TOKEN`
- Прописать их в проект

## Шаг 3 — Cloudflare Pages (или Netlify)
- Выбрать платформу (рекомендую Cloudflare Pages)
- Создать аккаунт (если нет)
- Подключить GitHub-репозиторий
- Настроить сборку:
  - Build command: `npm run build`
  - Output dir: `dist/`
- Задеплоить

## Шаг 4 — Домен
- После деплоя сайт будет доступен по бесплатному домену
- В любой момент можно подвязать свой домен (бесплатно)
